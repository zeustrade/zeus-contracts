# zeus-contracts

Smart contract for Zeus leverage DEX

- Language: Solidity v0.6.12

- Project framework: hardhat (ethers) + typechain

- Nodejs: v16.20.1


## Installation & Usage

1. Install packages
```
yarn
```

2. Build project
```
yarn build
```

### Testing

```
yarn test
```

### Run linter

```
yarn lint
```
